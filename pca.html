공정능력 마스터 매뉴얼 v1.1

1. 개요

  • Excel 방식: 표본 표준편차 (STDEV.S) 함수를 기반으로 전체 장기적 산포를 분석하여 Pp, Ppk 값을 산출합니다.
  • Minitab 방식: d2 계수표(군내 변동)를 기반으로 부분군의 단기 잠재력을 분석하여 Cp, Cpk 값을 산출합니다.

2. 사용방법

  • 규격 입력: 상한(USL), 하한(LSL)을 입력합니다. (군 크기 n은 기본값=1이며 필요 시 수정)
  • 데이터 업로드: 템플릿과 동일한 형태의 엑셀 파일(.xlsx)을 업로드합니다.
  • 분석 시작: 파란색 [분석 시작하기] 버튼을 누르면 실시간 대시보드가 생성됩니다.
  • 리포트 다운로드: 엑셀 리포트 버튼을 통해 결과를 출력할 수 있습니다.

3. 판정 기준

우수 >= 1.67
양호 >= 1.33
보통 >= 1.00
부족 < 1.00
